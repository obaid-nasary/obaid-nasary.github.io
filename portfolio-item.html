Employees Manager

manage with ease

Employees Management is web application based on role access. The application in the current state has 5 user roles with different authorities. A user with admin or manager role has the ability to add new users to the system, and update the users with the limitation of changing the user role which only super admin has the authority to change or delete a user. I am working on expanding it more, and bringing more functionalities into it.

The application is built in Spring Boot for the back-end and Angular with Bootstrap for front-end. The main purpose of doing this project was to get more familiar with Spring Security and Jason Web Token.

The application is also deployed in AWS and by clicking the SEE LIVE button below will navigate you to the live view of the application.

IBAN Number Validator

check IBAN numbers

It is a basic web application to check an IBAN number based on the rules set internationally. The application has the ability to check if a given number is valid or invalid and provides the user with update right while adding it to the database.

The application also has the ability to print log of recently added IBAN number and filter them based on valid and invalid IBAN numbers.

The application is built in Spring Boot for back-end, Angular for front-end and MySQL as for the database.

GIPHY GIFs Finder

find Gifs of choice

To be able to search for for more related GIFs and have accurate results, this application is using the IBM Natural Language Understanding API to extract the keywords with highest relevance and search for the GIF based on the keywords extracted.

The application uses the GIPHY trending API to showcase the trending GIFs of the day on the landing page, and also uses the search API to search for GIFs of the choice.

The application is built in Spring Boot for back-end, and Angular with basic Bootstrap for front-end.
